# 教学
## 剪刀石头布
伪代码：先给出3个变量，再获得用户的输入，电脑再从三个变量中随机选择一个

### 任务
1. 设计一个石头剪刀布游戏
2. input， if， 循环 break continue， = ==， and or的用法，字典和列表
3. if的优化（代码的重构）程序的健壮性 非法输入的处理
4. 自己想的优化点 电脑随机出拳（advantage：等概率直接取整，可以直接取数。

课后作业
电脑跟据我出拳的招式优化自己的策略

### 优点
快速找到自己的问题，能找到优化的点，随机出拳，策略出拳。
不要给自己设限制。
